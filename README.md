# 🎬 Entertainment Web App

A modern movie & TV series web application built with **Vue 3**, **TypeScript**, and **TanStack Query**.  
Users can browse content, search titles, and manage bookmarked movies.

---

## 🚀 Tech Stack

- **Vue 3 (Composition API)**
- **TypeScript**
- **Vite**
- **TanStack Query (Vue Query)** – server state management & caching
- **Axios** – HTTP requests
- **SCSS**

---

## ✨ Features

- Browse all movies & TV series
- Filter by category (Movie / TV Series)
- Real-time search
- Bookmark / remove bookmark
- Reactive UI updates without refetching
- Optimized caching with TanStack Query
- Responsive layout (mobile → desktop)
